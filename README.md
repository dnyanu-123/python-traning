# AI Trip Planner

AI Trip Planner is a web application built using Python Flask.

## Features

* User Registration and Login
* Traveller Management
* Trip Management
* Dashboard
* Search and Filters
* AI Itinerary Generation
* Admin Panel
* Responsive Design

## Technologies Used

* Python
* Flask
* SQLite
* HTML
* Bootstrap
* JavaScript
